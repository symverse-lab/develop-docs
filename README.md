# Symverse 소개

## 심버스는 자기주권형 분산ID를 기반으로 작동하는 1초 대의 블록확정성을 가진 멀티 블록체인 플랫폼 생태계입니다.

독자적인 ID체계를 통하여 심버스플랫폼 기반의 독립적인 블록체인플랫폼들은 상호 연동하는 협력적인 생태계로서 블록체인 서비스를 확장시킬 수 있습니다.
