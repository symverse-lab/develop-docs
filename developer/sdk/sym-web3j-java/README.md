# sym-web3j (JAVA)

**sym-web3는 Java 환경에서 Symverse 블록체인의 Dapp 개발을 지원해주는 SDK입니다.**

{% hint style="info" %}
기본적으로 이더리움의 Web3J를 Fork하여 Symverse에서 고유로 사용하는 Hash 함수 및 API 를 추가하여 Wrap한 형태로써, 기본함수는 네임스페이스만 변경되었고(eth -> sym) 함수명은 일치합니다.
{% endhint %}
